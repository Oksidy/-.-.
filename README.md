1. Общие сведения
Название продукта: Программное обеспечение для автоматизации расчётов в оптике (АС-ОПТИКА)
Разработчик: Салмин А.А.
Научный руководитель: Михайличенко Ж.В.
Организация: Орский гуманитарно-технологический институт (филиал) ОГУ

2. Описание продукта
Программное обеспечение предназначено для автоматизации расчётов в области оптики. Программа позволяет выполнять:
* Расчёт углов преломления и отражения
* Вычисление параметров линз
* Анализ работы зеркал
* Расчёт интерференционных и дифракционных явлений

3. Системные требования
* Операционная система: Windows 10/11
* Платформа: .NET 6.0 Desktop Runtime
* Процессор: 1 ГГц или выше
* Оперативная память: 2 ГБ
* Место на диске: 500 МБ

4. Установка
1. Скачайте архив с программой
2. Распакуйте содержимое архива
3. Запустите файл AS-OPTIKA.exe
4. При необходимости установите .NET 6.0
